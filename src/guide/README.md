## 戴森球计划知识库介绍

戴森球计划知识库 | Dyson Sphere Program Knowledge Base (DSP-KB)  
主要用于收集、整理、分享戴森球计划相关的知识和信息。  
知识库内容包括：游戏机制、专项研究、常见问题解答、种子与蓝图、模组与工具等方面。  
由 DSP 爱好者、游戏玩家、程序员共同维护，欢迎各位加入我们一起建设 DSP-KB！

## 参与贡献
