## 一般设置
| 设置项 | 可选项 | 释义 |
| :--- | :-: | :-: |
| 星区种子 | 八位十进制数组成 | 不同的种子会得到不同的星图 [种子推荐](/blueprints/seed) |
| 恒星数量 | 32 ~ 64 | 星图中恒星的数量 |
| 资源倍率 | 极少 (0.1) ~ 无限 |  |
| 沙盒模式 | - | 创造模式，无法获得任何元数据、无法解锁任何成就 |
